# Screenshots

Guarda las imágenes del proyecto en esta carpeta con los siguientes nombres:

1. **settings.png** - Modal de "Ajustes de Flujo"
2. **focus-mode.png** - Modo Enfoque con burbujas de tareas flotantes
3. **calendar-view.png** - Vista de calendario (Marzo 2026)
4. **list-view.png** - Vista de lista con "Todas las tareas"
5. **kanban-view.png** - Vista Kanban con columnas (Pendientes, Por hacer hoy, En proceso, Hecho)

## Cómo añadir las imágenes

### Opción 1: Copiar manualmente
Arrastra las 5 imágenes que compartiste a esta carpeta y renómbralas según la lista de arriba.

### Opción 2: Desde tus archivos
Las imágenes adjuntas en el chat corresponden a:
- Imagen 1 → **settings.png**
- Imagen 2 → **calendar-view.png**
- Imagen 3 → **list-view.png**
- Imagen 4 → **kanban-view.png**
- Imagen 5 → **focus-mode.png**

Una vez copiadas, el README mostrará automáticamente todas las capturas de pantalla.
